# つぎへ (Tsugi e)

最聪明的花火导航。

つぎへ 是一款基于时间与地理位置的移动应用，核心目标是：
在「当前时间 + 当前地点」条件下，给你下一场值得去的花火活动。

## 核心体验

- 打开即用，不需要登录
- 首屏就是地图，直接看到附近活动
- 自动给出“今日推荐”，减少选择成本
- 点开活动详情后可一键导航出发

## 主要功能

### 1. 首页地图
- 自动定位当前所在位置
- 在地图上展示附近活动标记
- 进入首页默认全屏地图，不自动弹卡片

### 2. 今日推荐
- 系统只给出一个当前最值得去的活动
- 卡片展示：活动名称、距离、开始时间、倒计时、简短推荐语

### 3. 地图交互
- 点击地图标记可查看“快速查看”半屏卡片
- 上滑卡片进入活动详情页

### 4. 活动详情
- 展示活动名称、日期时间、距离、规模、最近车站、推荐语
- 提供一键导航按钮，快速前往

### 5. 时间筛选
- `Today`
- `This Week`

### 6. 开始前提醒
- 支持本地通知提醒
- 无需账号即可设置提醒

## 推荐逻辑（MVP）

当前推荐优先级：

1. 今日优先
2. 距离优先
3. 规模优先

## 适用人群

- 想临时决定今晚去哪里看花火的人
- 旅游中希望快速判断“下一场活动”的用户
- 不想花时间筛选信息、希望直接获得建议的人

## 当前阶段边界（MVP）

当前版本聚焦“快速决策 + 快速出发”，以下能力暂不包含：

- 账号与登录
- 社交与评论
- 复杂个性化算法
- 支付与订阅
